== Description ==

Displays monthly archives along with post titles and optional post excerpts

== Installation ==

Unzip the DM-Archives folder into the plugins folder or Use the wordpress admin panel to install new plugins and then select the zip folder

== Usage ==

* Type [dm-archives] in a page
* To display post excerpts, change the settings to true in the Settings panel
